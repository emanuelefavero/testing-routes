# Testing - Routes (Express, Jest, SuperTest)

This is a simple example of how to test routes with Express, Jest and SuperTest NPM testing library.

## Installation

```bash
npm install
```

## Usage

```bash
npm run test
```

## Needed Dependencies and settings

```bash
npm install express
npm install jest
npm install supertest
```

- package.json

```json
"scripts": {
    "test": "jest --watch"
  },
```

- routes file

```bash
routes.js
```

- test file

```bash
routes.test.js
```
